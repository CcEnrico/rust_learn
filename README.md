# 📚 1. Rust Fundamentals (PRIMA DI GAMEDEV)

## ⭐ The Rust Programming Language ("The Book")

Link:
https://doc.rust-lang.org/book/
Interactive version:
https://rust-book.cs.brown.edu/print.html

doc

```bash
rustup doc  
```

book

```bash
rustup doc --book 
```

per aprire la doc delle dipendenze "crates"

```bash
rustup doc --open 
```

Perché:

- Risorsa ufficiale
- Gratis e open source
- Fondamentale per capire ownership e borrowing

Studiare almeno:

- Ownership
- Struct e Enum
- Pattern matching
- Error handling (Result / Option)
- Collections

1. Rust Book (prima metà)
2. Rustlings exercises
3. Rust By Example (consultazione)

# 🚫 Errori da evitare

- Non partire subito con engine complessi
- Non fare un progetto troppo grande
- Non ignorare ownership e borrowing
